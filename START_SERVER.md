# Как запустить dev сервер

## Быстрый запуск

Откройте терминал в папке проекта и выполните:

```bash
npm run dev
```

## Что должно произойти

После запуска вы увидите что-то вроде:

```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help
```

## Откройте браузер

Скопируйте адрес из терминала (обычно `http://localhost:5173/`) и откройте его в браузере.

## Если порт занят

Если порт 5173 занят, Vite автоматически предложит использовать другой порт (например, 5174).

Следуйте инструкциям в терминале.

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.

## Проблемы

### Ошибка "port already in use"

Порт уже занят другим процессом. Решения:

1. **Используйте другой порт:**
   ```bash
   npm run dev -- --port 3000
   ```

2. **Или найдите и остановите процесс на порту 5173:**
   ```bash
   # Windows
   netstat -ano | findstr :5173
   # Затем найдите PID и остановите процесс
   taskkill /PID <PID> /F
   ```

### Ошибка "Cannot find module"

Установите зависимости:
```bash
npm install
```

### Сервер не запускается

1. Проверьте, что Node.js установлен:
   ```bash
   node --version
   npm --version
   ```

2. Переустановите зависимости:
   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```

3. Попробуйте запустить снова:
   ```bash
   npm run dev
   ```

## Проверка работы

После запуска сервера:

1. Откройте браузер
2. Перейдите по адресу из терминала (обычно `http://localhost:5173/`)
3. Вы должны увидеть стартовый экран игры Тамагочи

## Важно

- **Не закрывайте терминал** - сервер должен работать постоянно
- **Сохраняйте изменения** - Vite автоматически перезагрузит страницу при изменении файлов
- **Проверяйте консоль браузера** - там могут быть ошибки (F12)
